# CareClick: Universal Healthcare Scheduling

## The Problem
Many elderly patients struggle with complex healthcare portals or long wait times on phone calls, creating a digital barrier to essential medical services.

## The Solution
A simplified, 5-step linear booking system designed specifically for accessibility.

### Key Features for Accessibility:
* **High-Contrast UI:** Optimized for users with visual impairments.
* **Minimalist Navigation:** No complex menus; only "Back" and "Next" buttons.
* **Large Touch Targets:** Buttons and forms designed for easy interaction.
* **Visual Progress:** A clear "Step X of 5" indicator to reduce anxiety.

## Tech Stack
* **Frontend:** React.js / Tailwind CSS
* **Deployment:** Vercel / GitHub Pages
